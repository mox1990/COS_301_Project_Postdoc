Application services menu

Please select the service you wish to use

This option allows you to view the progress and information attachments of applications for which you are the fellow or grant holder of.

Further if you are part of DRIS you can use this option to view the above for all applications on the system. This option allows you to create progress reports for any fellowships which you are the fellow of. This option allows you to apply for a fellowship. Please use it to open an new application. For renewals please use the renewal option below. This option allows you to renew any applications which you are the fellow of which qualify for renewals. This option allows you create referral reports for those submitted new fellowship application for which you have been requested to referee. This option allows you to review and finalise any referred new or renewal fellowship applications for which you are the grant holder of. This option allows you to review, recommend, decline or request the ammendment of any finalised new or renewal fellowship applications that are pending for your approval. This option allows you to review, endorse or decline any recommended new or renewal fellowship application. This option allows you to review, decline, check the eligiblity of and approve or edit funding details for any endorsed new or renewal fellowship application. This options allows you to schedule, manage, comment on and review any post doctoral committee meetings. This option allows you to forward or rewind any application still in the application pipeline through the various stages. Note the use of this service can cause the irrevocable loss of information regarding any selected application.
